# MERN Chat Starter (zip)
        This is a compact MERN starter for a chat app (authentication, chats, messages, socket.io).

## Quickstart (local)
        1. Install MongoDB and start it locally, or use Atlas and set MONGO_URI in backend/.env
        2. In `backend/` run: `npm install` then `npm run dev`
        3. In `frontend/` run: `npm install` then `npm start`
        4. Copy `.env.example` to `.env` in both backend and frontend and fill values.

This zip is a starter — you may want to add Cloudinary keys, polish UI, and add production configs.
